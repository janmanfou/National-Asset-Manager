Create a production-ready web application named **National Voter PDF to Excel Converter**.

Purpose:
Upload scanned voter list PDFs (All India level), process them one by one, extract structured voter data using OCR, and export clean Excel files.

### Core Features

1. **Batch PDF Upload**

   * Upload multiple scanned PDFs
   * Queue-based processing (one by one automatic processing)
   * Show upload status: pending / processing / completed / failed

2. **Advanced OCR Engine**

   * Extract text from scanned images inside PDFs
   * Support Hindi + English
   * Preprocessing: noise removal, contrast enhance, auto-rotate
   * Detect tabular voter data layout

3. **Structured Data Extraction**
   Extract fields if available:

   * Voter Name
   * Father/Husband Name
   * Gender
   * Age
   * Address
   * Booth Number
   * Part Number
   * Serial Number
   * Constituency
   * State
   * EPIC Number

4. **Data Cleaning & Standardization**

   * Remove duplicates
   * Normalize gender values
   * Fix broken text rows
   * Validate age and number fields

5. **Excel Export System**

   * Auto-generate .xlsx file per PDF
   * Download button
   * Bulk download option
   * Column mapping configurable

6. **Processing Dashboard**

   * Total files uploaded
   * Completed conversions
   * Failed files
   * Processing progress bar
   * Processing logs viewer

7. **API Integration Ready**

   * Provide API endpoint to send processed Excel to main election system
   * Secure token-based API

8. **Performance Architecture**

   * Background worker for OCR processing
   * Queue system
   * Memory safe processing for large PDFs

9. **User Interface**

   * Clean admin panel
   * Drag & drop upload
   * File history
   * Download center

10. **Security**

* Admin login
* File size limits
* Secure file storage
* Auto-delete raw files after processing (optional setting)

### Tech Requirements

* Python backend
* OCR engine integration
* Excel generation module
* Modular architecture
* Production-ready code structure

### Output Format

Each processed PDF should generate:

* Clean Excel file
* Structured voter table
* Processing report

Name the system:
**Voter Data Conversion Engine**
